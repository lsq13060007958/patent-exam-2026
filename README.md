# 专利代理师考试 2026

2026年专利代理师资格考试刷题助手 — 响应式暗色设计版

## 功能特点

- 📚 **专利法 669 题** — 30 个专题全覆盖
- ⚖️ **相关法 829 题** — 10 个专题全覆盖
- 📘 **专题讲座** — 韩龙老师讲座原文，含思维导图
- 📱 **响应式设计** — PC 侧边栏 + 移动端 TabBar，可手动切换
- 🌙 **暗色主题** — Linear 风格设计
- 💾 **本地存储** — 做题进度自动保存

## 使用方法

直接访问 GitHub Pages 即可使用：

**https://lsq13060007958.github.io/patent-exam-2026/**

## 设备切换

- **自动模式**：根据屏幕宽度自动切换 PC/移动端布局
- **手机模式**：强制使用移动端 TabBar 布局
- **电脑模式**：强制使用 PC 侧边栏布局

切换按钮位于：
- PC 端：左侧边栏底部
- 移动端：右上角浮动按钮

## 技术栈

- 纯 HTML/CSS/JavaScript，无框架依赖
- CSS 响应式布局（media query）
- localStorage 本地数据持久化
